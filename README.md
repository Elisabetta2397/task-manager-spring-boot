# 🚀 Task Manager

Task Manager è un'applicazione web sviluppata con **Java** e **Spring Boot** per la gestione di attività personali.

## 🛠️ Tecnologie utilizzate

- Java 21
- Spring Boot 3.5.15
- Spring Security
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap 5
- Maven
- Git & GitHub

## ✨ Funzionalità implementate

- ✅ Homepage
- ✅ Connessione al database MySQL
- ✅ Persistenza dati con Spring Data JPA
- ✅ Registrazione utenti
- ✅ Validazione dei campi del form
- ✅ Password criptate con BCrypt
- ✅ Controllo email duplicata
- ✅ Gestione degli errori di registrazione
- ✅ Login utenti
- ✅ Logout
- ✅ Dashboard personale
- ✅ Creazione di un task
- ✅ Modifica di un task
- ✅ Eliminazione di un task
- ✅ Gestione dello stato dei task
- ✅ Gestione della priorità dei task
- ✅ Visualizzazione dei task dell'utente autenticato
- ✅ Filtro per stato
- ✅ Filtro per priorità
- ✅ Filtro combinato per stato e priorità



---

## 🚧 Funzionalità in sviluppo

- 🔄 Ricerca Task
- 🔄 Gestione ruoli (USER / ADMIN)

## ▶️ Avvio del progetto

1. Clonare il repository

```bash
git clone https://github.com/Elisabetta2397/task-manager-spring-boot.git
```

2. Creare un database MySQL chiamato `taskmanager`

3. Configurare il file `application.properties`

4. Avviare l'applicazione

```bash
./mvnw spring-boot:run
```
